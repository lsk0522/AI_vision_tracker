# 릴리즈 노트 (Release Notes)

이 문서에는 AI Vision Tracker 프로젝트의 버전별 추가, 변경, 삭제된 기능들이 타임라인 형식으로 기록됩니다.

---

## [v1.3.0] - 2026-07-03
### 🚀 추가 (Added)
- **라즈베리파이(Raspberry Pi) 전용 환경 구축:** 윈도우 의존성을 벗어나 라즈베리파이 터미널(Headless CLI) 환경에서 완벽하게 구동되는 `raspi_main` 독립 패키지 신설.
- **자동화 스크립트 3종:** 복잡한 명령어 없이 한 번에 구동 가능한 `install.sh`, `start.sh`, `update.sh` 추가.
- **CLI 로딩 애니메이션:** 터미널 부팅 시 `rich` 라이브러리를 활용한 직관적이고 세련된 핑크색 도트 스피너(`cli_ui.py`) 추가.
- **프로덕션 서버 연동:** Flask 기본 개발 서버 대신, 접속 안정성이 뛰어난 `waitress` WSGI 서버를 라즈베리파이 환경에 자동 연동.

### 🛠 변경 및 개선 (Changed & Fixed)
- **Linux(V4L2) 카메라 프레임 최적화:** 라즈베리파이에서 고해상도(1080p) 고프레임(30fps) 송출을 위해 `cv2.CAP_V4L2` 캡처 백엔드에 `MJPG` 코덱 강제 할당 로직 추가 (`camera.py`).
- **중복 리소스 싹둑 정리:** 이전 버전에서 발생한 쓸모없는 백업 파일들 및 외부 압축 해제 찌꺼기 중복 폴더 완전히 청소.

## [v1.2.0] - 2026-07-02
### 🚀 추가 (Added)
- **Apple Design System 적용:** 불필요한 그림자와 입체감을 모두 제거하고, 단일 포인트 컬러(Action Blue)를 사용한 극도의 미니멀리즘 플랫(Flat) UI로 전면 개편.
- **펌웨어 버전 불일치 강제 차단(Hard Crash):** 연결된 ESP32 펌웨어가 파이썬 백엔드가 기대하는 버전(2.0.7)과 다를 경우, 단순 경고를 넘어 파이썬 프로세스를 즉시 종료(os._exit(1))하여 예기치 않은 하드웨어 오작동을 원천 차단.

### 🛠 변경 및 개선 (Changed & Fixed)
- **조이스틱 뚝뚝 끊김(Stuttering) 현상 완벽 해결:**
  - 웹 조이스틱의 HTTP 갱신 주기를 20ms에서 80ms로 늦추고 불안정한 패킷 취소(Abort) 로직을 제거하여 통신 안정성 확보.
  - ESP32 펌웨어의 조그(JOG) 안전 정지(Watchdog) 타이머를 150ms에서 350ms로 늘려 무선 지연(Jitter)으로 인한 급정거 현상 방지.

## [v1.1.0] - 2026-06-30
### 🚀 추가 (Added)
- **미디어 라이브러리 이미지 삭제 기능:** 미디어 갤러리에 저장된 이미지를 휴지통 아이콘으로 개별 삭제할 수 있는 API 및 UI 추가.
- **Arduino 폴백(Fallback) 지원:** motor_arduino.py를 추가하여 ESP32가 아닌 일반 아두이노(Uno 등) 환경에서도 동작할 수 있는 폴백 뼈대 구성.

### 🛠 변경 및 개선 (Changed & Fixed)
- **UI 레이아웃 정리 및 최적화:**
  - 모터 파라미터 제어 탭 이름을 각각 📷 자동 추적 설정, 📐 수동 좌표 제어로 직관적으로 변경.
  - 불필요한 게이지바와 텍스트를 제거하고, 필수 제어 버튼(Set Home, E-STOP)을 화면 하단 좌/우측 모서리(Floating Absolute)로 깔끔하게 분리 배치.
  - 파라미터 설정 변수명 직관화 (추적 스텝 비율, 추적 최대 속도 등).

## [v1.0.0] - Initial Release
### 🚀 추가 (Added)
- **AI 추적 코어:** OpenCV CSRT Tracker, Competitive Tracking (전역 템플릿 매칭), Kalman Filter 융합 다중 추적 알고리즘 탑재.
- **ESP32 모터 제어 펌웨어:** FreeRTOS 듀얼코어 기반 타이머 인터럽트(10ms 주기) P-Control 스텝모터 가속도 제어.
- **Glassmorphic Web UI:** PC 및 모바일 반응형 웹 인터페이스, 실시간 MJPEG 비디오 스트리밍 출력 기능.
- **동적 파라미터 제어:** 웹 브라우저에서 실시간으로 스테퍼 모터의 최대 속도, 가속도, steps/px 값을 설정하여 보드로 동기화.
