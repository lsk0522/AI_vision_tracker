# 🏆 AI Vision Tracker v1.4.0 Release Notes

이번 v1.4.0 릴리즈는 **하드웨어 보호와 정밀한 움직임 제어**에 초점을 맞춘 대규모 안정성 업데이트입니다. 실제 기기의 물리적 한계를 소프트웨어가 완벽히 이해하고, 충돌이나 오버슛(밀림) 현상 없이 부드럽고 안전하게 작동하도록 모든 리밋 시스템을 재설계했습니다.

---

## ✨ 주요 업데이트 내용

### 1. 📏 하드웨어 맞춤형 물리 각도 실측 적용
단순한 기어비(1:5) 계산식을 넘어, 모터 스텝수와 실제 하드웨어의 가동 범위를 실측하여 완벽하게 맵핑했습니다.
- **M1 (수평, Pan)**: `±180도` (총 360도 회전)
- **M2 (수직, Tilt)**: 위로 `+50도`, 아래로 `-40도` (총 90도 회전)

### 2. 🛡️ 다이나믹 2D 충돌 방지 안전 구역 (Safe Zone)
카메라 렌즈가 아래를 향할 때 베이스(기둥) 구조물과 부딪히는 하드웨어 파손을 원천 차단하는 지능형 안전 로직을 탑재했습니다.
- M2(수직) 각도가 `-25도`보다 아래로 내려가면, 즉시 M1(수평)의 회전 반경을 **±180도에서 ±80도로 자동 축소**시킵니다.
- 기기가 스스로 구석에 갇히는 현상을 방지하기 위해, M1이 안전 구역 밖에 있을 때는 M2가 아래로 내려가는 것을 소프트웨어가 선제적으로 차단합니다.

### 3. 🏎️ 자동 감속 제동 시스템 (Soft Braking) 탑재
빠른 속도로 조이스틱을 조작하다가 멈출 때 발생하는 관성 밀림(Overshoot) 현상을 자동차의 긴급 제동 시스템(AEB)처럼 해결했습니다.
- 리밋 한계선 도달 전 (M1은 20도 전, M2는 15도 전)부터 **자동으로 조이스틱 입력값을 부드럽게 감속**시킵니다.
- 사용자가 풀악셀(최대 속도)로 조작하더라도 한계선에서는 속도가 `0`으로 수렴하여, 하드웨어 충격이나 선 넘김 없이 칼같이 멈춥니다.

### 4. 🚨 터미널(TUI) 실시간 리밋 경고창 지원
라즈베리파이 백엔드 터미널(대시보드)에서 모터가 왜 멈췄는지 직관적으로 알려줍니다.
- 정상 작동 중에는 `[ON]`, 입력이 차단되었을 때는 빨간색 `[BLOCKED]`로 표시됩니다.
- 멈춘 원인을 실시간으로 분석해 **"⚠️ [M2 DOWN] 2D Safe Zone (-25°)"** 또는 **"⚠️ [M1 LEFT] Hardware Limit (-180°)"** 와 같이 구체적인 사유를 출력합니다.

### 5. 📱 웹 UI 전체화면(Fullscreen) 모드
이제 스마트폰, 태블릿, PC 등 다양한 환경에서 브라우저 주소창 방해 없이 앱처럼 꽉 찬 화면으로 쾌적한 조작이 가능합니다.
- 우측 상단 `[🔲 전체화면]` 버튼을 터치하여 원클릭으로 화면을 확장/축소할 수 있습니다.
