# 🎯 AI Vision Tracker (Turret)

> **스마트 객체 추적 시스템 및 웹 기반 터렛 컨트롤러**  
> 고성능 객체 인식(CSRT) 알고리즘과 ESP32 스테퍼 모터 정밀 제어를 통해 움직이는 객체를 자동으로 추적하고 모니터링하는 AI 비전 트래커 프로젝트입니다.

---

## ✨ 주요 기능 및 특징 (Features)

### 🖥️ Apple 스타일 Glassmorphic Web UI & UX
- **다이내믹 인터페이스**: 모바일 친화적인 반응형 레이아웃 및 부드러운 애니메이션 적용.
- **직관적인 조작**: 가상 조이스틱(수동 제어), 갤러리 뷰어, 실시간 설정 패널 제공.
- **상태 모니터링**: `LOCKED`, `SEARCHING` 등 실시간 추적 상태 및 모터 연결 상태 시각화.

### 🤖 지능형 객체 추적 알고리즘 (AI Tracking)
- **CSRT 기반 객체 추적**: 드래그 한 번으로 대상을 지정하고 추적.
- **Competitive Tracking (경쟁적 추적 알고리즘)**: 화면을 벗어났거나 놓친 타겟을 전역 템플릿 매칭을 통해 스스로 다시 찾아내는 스마트 복구 로직 적용.
- **다중 추적 지원**: 움직임 감지, 색상 감지 및 형태(원형) 감지 폴백 시스템 구축.

### ⚙️ 고성능 하드웨어 제어 (ESP32 & 스텝 모터)
- **부드러운 정밀 가속 (P-Control)**: 0 → 3000Hz 속도까지 8.0Hz/ms의 가속도로 부드럽고 강력하게 이동.
- **하드웨어 튜닝**: 1:5 기어비 (모터 16T / 출력 80T) 지원.
- **실시간 파라미터 동기화**: 웹 UI에서 설정한 가속도, 최대 속도, 픽셀당 이동 거리가 즉각적으로 ESP32 컨트롤러에 반영됨.
- **FOTA(Firmware Over-The-Air) 업로드**: 웹 UI에서 버튼 한 번으로 ESP32 펌웨어 컴파일 및 업로드 지원 (arduino-cli 연동).

## 🏗️ 시스템 아키텍처 (Architecture)

### 1. 백엔드 (Python / Flask)
- `main.py`, `routes.py`: API 서버, 웹 페이지 렌더링 및 모터/카메라 스레드 통합.
- `detector.py`: OpenCV 기반 영상 처리 및 추적, Kalman 필터를 이용한 좌표 예측.
- `camera.py`: 웹캠 캡처 및 비디오 스트림 파이프라인 관리.

### 2. 하드웨어 제어 (Python + C++)
- `motor_esp32.py`: Python 서버와 ESP32 간 시리얼 통신 브릿지 (비동기 명령어 큐잉 구조).
- `esp32_firmware/`: 하드웨어 타이머 인터럽트를 사용해 DM542 드라이버를 빠르고 정밀하게 병렬 제어하는 C++ 펌웨어.

### 3. 프론트엔드 (HTML/CSS/JS)
- 모던 웹 표준 기술만 사용하여 별도의 무거운 프레임워크(React/Vue 등) 없이 초경량/고성능 Glassmorphism 인터페이스 구현.

## 🚀 설치 및 실행 방법 (How to Run)

### 1. 요구 사항
- Python 3.11+
- Arduino CLI (펌웨어 업로드 사용 시)
- 연결된 웹캠 및 ESP32 + 스텝 모터 드라이버(DM542)

### 2. 의존성 설치
```bash
pip install flask opencv-python opencv-contrib-python numpy pyserial
```

### 3. 서버 실행
```bash
python main.py
```

### 4. 접속
- 로컬 환경: `http://localhost:5000`
- 모바일 및 동일 네트워크: `http://<호스트IP>:5000`

## 🛠️ 트러블슈팅 및 튜닝

- **추적이 잘 풀릴 경우**: 웹 설정의 '조준점 반응 속도'를 줄이거나 '객체 학습'을 다시 시도해주세요.
- **모터가 너무 느린 경우**: 기기 설정 > 모터 튜닝에서 최대 속도(Hz) 및 가속도를 높일 수 있습니다. (기본 최대속도: 3000Hz)
- **컴파일/타입 오류**: 본 프로젝트는 Pyright 기준 0 Error를 달성하여 완벽한 Type Hinting을 제공합니다. 오류 발생 시 Python 버전과 패키지를 확인하세요.
